# Token Reference

Master map of every design token. Source of truth is `app/tokens.css`; this
table is the lookup — when you need "the muted text color", find the role here,
never guess a value. Layer 1 is Tailwind v4's palette (from `@import "tailwindcss"`).
Palette: **slate neutrals + teal accent**, amber = warn, red = danger.

Utilities are generated by the `@theme inline` block: a token registered as
`--color-X` yields `bg-X`, `text-X`, `border-X`, `divide-X`, `ring-X`, `accent-X`, …

## Removed in the 2026-07 consolidation

`ink-soft`→`ink-muted` · `ink-faint`→`ink-subtle` · `surface-header`→`surface` ·
`surface-inset-strong`→`surface-inset` · `line-hover`/`line-focus`→`line-active` ·
`info-*`→ neutral notice (`border-line bg-surface-subtle text-ink-muted`).
Do not re-add without a real second use case.
